# IMDb Analytics Dashboard

Next.js (App Router) dashboard that consumes the aggregated outputs generated by the pipeline in `public/data`.

## Run locally

```bash
cd dashboard
npm install
npm run dev
```

Open `http://localhost:3000`.

## Data

The dashboard reads only JSON/CSV files in `public/data`. Run the pipeline before starting the app.

## Deploy

Compatible with Vercel. Set the `dashboard/` directory as the project root.

## Language

Use the language switcher in the top navigation to toggle between English and pt-BR.
